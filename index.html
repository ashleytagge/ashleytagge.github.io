Projects

Datalog Program

Developed a series of C++ programs that progressively built a full Datalog interpreter. Starting with a lexical scanner to tokenize input, I implemented a parser to construct syntax trees, designed a relational database engine to evaluate facts and queries, and expanded it to handle logical rules. The final phase focused on optimizing rule evaluation for faster, more efficient query processing, combining compiler design, database logic, and algorithmic problem-solving in one cohesive system.

Fine Motor Skills Application

Built a Java application designed to help children develop fine motor skills through fun, drag-and-drop activities. The app features colorful visuals and intuitive interactions that encourage hand–eye coordination and precision, blending playful design with educational purpose.

AquaQuest Web Application

Created a simple web app where paddleboarders and boaters can check in when visiting national parks. Each check-in fills a personal progress bar, letting users share their outdoor adventures and celebrate how active they are in nature. Designed to make tracking outdoor activity fun, social, and motivating.
